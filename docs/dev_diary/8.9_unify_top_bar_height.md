# 8.9 — Унифицирована высота всех шапок в UI

**Дата:** 2026-04-24
**Этап:** 8.9
**Коммит:** `35dfaff`

## Что сделано

Верхняя шапка приложения (`AppShell.tsx`) была `h-11` (44px), внутренние шапки (шапка поиска диалогов и шапка конкретного диалога) — `h-12` (48px) после 8.8. Подтянул AppShell к тому же `h-12`. Теперь по всей странице шапки одной высоты, без 4px-ступеньки.

## Файлы

- [modules/ui_web/src/components/layout/AppShell.tsx:25](../../modules/ui_web/src/components/layout/AppShell.tsx#L25) — `h-11` → `h-12`.

## Почему h-12, а не h-11

Две из трёх шапок уже были `h-12` (фиксировал в 8.8). Проще подтянуть одну (AppShell) чем откатывать две. Плюс h-12 (48px) — симпатичная круглая величина, не требующая точной подгонки текста/иконок внутри.

## Проверка

Визуально — сплошная линия бордера между AppShell и следующей строкой (карточки аккаунтов), потом ровная линия между поиском диалогов и ледой сообщений. Никаких мелких ступенек на границах секций.
