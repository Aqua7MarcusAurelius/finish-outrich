## Структура БД (`database_schema.md`)

**Новый документ.** Создан при финальной реструктуризации — ранее структура БД была размазана по `module_history_v5.md` (6 таблиц истории) и `configuration_v1.md` (таблица `settings`), а таблица архива событий шины вообще не имела описания хотя упоминалась в `event_bus_v3.md`.

Содержит все таблицы PostgreSQL в одном месте:
- `accounts`, `dialogs`, `messages`, `media`, `reactions`, `message_edits` — перенесены из модуля истории
- `settings` — перенесена из конфигурации
- `events_archive` — новая, описана впервые (архив шины событий)
